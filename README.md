## 🐛 Troubleshooting

### Website Won't Load
**Solution:** Ensure all 3 files (index.html, style.css, script.js) are in the same folder

### Styles Don't Apply
**Solution:** Hard refresh browser (Ctrl+Shift+R) or clear cache

### JavaScript Not Working
**Solution:** 
- Open Developer Console (F12)
- Check for red errors
- Ensure script.js is in same folder

### Dark Mode Doesn't Persist
**Solution:** 
- Clear browser cache
- Check if localStorage is enabled
- Try incognito mode

### Form Validation Issues
**Solution:**
- Check browser console for errors
- Verify all input IDs match in HTML and JS
- Test on different browsers

## 📚 Code Examples

### Dynamic Counter Animation
```javascript
function animateCounter(id, target) {
    let count = 0;
    const increment = Math.ceil(target / 100);
    const interval = setInterval(() => {
        count += increment;
        if (count >= target) {
            count = target;
            clearInterval(interval);
        }
        document.getElementById(id).innerText = count.toLocaleString();
    }, 20);
}
```

### Form Validation
```javascript
const emailPattern = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;
if (!emailPattern.test(email)) {
    showError("emailError", "Please enter a valid email address");
    isValid = false;
}
```

### Theme Toggle with Storage
```javascript
themeBtn.addEventListener("click", () => {
    document.body.classList.toggle("dark-mode");
    const theme = document.body.classList.contains("dark-mode") ? "dark" : "light";
    localStorage.setItem("theme", theme);
});
```

### Image Slider
```javascript
function changeSlide(n) {
    currentSlideIndex += n;
    if (currentSlideIndex >= slides.length) currentSlideIndex = 0;
    if (currentSlideIndex < 0) currentSlideIndex = slides.length - 1;
    
    slides.forEach(slide => slide.classList.remove("active"));
    slides[currentSlideIndex].classList.add("active");
}

setInterval(() => changeSlide(1), 5000); // Auto-advance
```
